# elixir-ls

ElixirLS - the Elixir Language Server and Debugger for Claude Code.

## Features

- Autocomplete and go-to-definition
- Inline diagnostics and warnings
- Documentation on hover
- Find references
- Code formatting
- Debugger support

## Supported Extensions

`.ex`, `.exs`, `.heex`, `.eex`

## Prerequisites

Install ElixirLS:

```bash
# Using Homebrew (recommended)
brew install elixir-ls
```

> Don't have Homebrew? Install it from [brew.sh](https://brew.sh)

**Note:** This will also install Elixir as a dependency if not already installed.

## Installation

```bash
# Add the marketplace (one time)
/plugin marketplace add ajanraj/claude-lsp-plugins

# Install the plugin
/plugin install elixir-ls@claude-lsp-plugins
```
